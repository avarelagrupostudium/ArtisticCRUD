       Tener instalada una máquina virtual de java. 

En caso de no tener una máquina podremos descargarla desde el siguiente enlace: 

https://www.java.com/es/download/


Tener instalado MySQL Workbench. 

En caso de no tener instalado el MySQL podremos descargarlo 

desde el siguiente enlace: https://dev.mysql.com/downloads/workbench/ .

 

En caso de tener algún problema en su instalación o configuración puedes consultar el manual 

de usuario de MySQL Workbench en https://dev.mysql.com/doc/workbench/en/
