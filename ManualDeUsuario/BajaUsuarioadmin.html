Entraremos en la pestaña de Baja para eliminar un usuario.


                                         


       



Aparecerá un desplegable con los usuarios donde podremos seleccionar el usuario que deseemos eliminar:



                                        



Una vez seleccionado el usuario que deseemos eliminar pulsamos el botón eliminar.




Nos aparecerá un mensaje preguntándonos si deseamos borrar el usuario y deberemos 

pulsa “Sí” para confirmar el borrado del usuario o “No” para cancelar la eliminación del usuario.


En caso de confirmar el borrado de usuario y este se realice correctamente saltará el 

siguiente mensaje:


                                                  



Si recibimos un error al borrar un usuario será porque es un artista y deberá ser eliminado 

desde la sección de baja artistas.
