Entraremos en la pestaña de Nuevo para dar de alta a un usuario.




Nos encontraremos una pantalla con diversos campos de texto. Para poder generar un 

usuario correctamente debemos rellenar todos los campos de texto y que la contraseña 

coincida en los dos campos.


Debemos seleccionar si el usuario es administrador o no, para ello seleccionaremos la circunferencia “Sí” para que el usuario sea administrador o “No” para que el usuario sea normal.
